# HaGOLEM WIKI - QUICK START SUMMARY

## What You're Getting:

A **wiki-like system** to organize all your HaGOLEM ideas from scattered files into one searchable database.

## How It Works (Simple Explanation):

### USER EXPERIENCE:
1. **List View** - See all idea titles with badges (category, type, views, comments)
2. **Click title** ▶ Shows summary (2-3 sentences)
3. **Click "Read More"** ▶ Opens full idea with complete content + all comments
4. **Add comments** - You and others can discuss, suggest changes, ask questions
5. **Search & filter** - Find ideas by keyword, category, or type

### FEATURES:
✓ **Three-level hierarchy**: Title → Summary → Full Content
✓ **Comments system**: Users can comment, suggest, question, approve
✓ **View tracking**: See which ideas are most viewed
✓ **Search**: Find ideas by keyword in title, summary, or content
✓ **Categories & Tags**: Organize ideas your way
✓ **Source tracking**: Remember which file each idea came from
✓ **Version tracking**: See how ideas evolve over time

## Files You're Getting:

1. **hagolem_wiki_schema.sql** - Database structure (3 tables: ideas, comments, history)
2. **hagolem_wiki_routes.py** - Backend Python code (API routes for Flask)
3. **hagolem_wiki.html** - Frontend interface (what users see)
4. **import_ideas.py** - Helper script to bulk import your existing ideas
5. **INSTALLATION_GUIDE.txt** - Step-by-step setup instructions
6. **THIS FILE** - Quick overview

## Installation (5 Steps):

1. **Create database tables** - Run the SQL file in MySQL
2. **Update database credentials** - Put your MySQL info in the Python file
3. **Add routes to Flask app** - Copy Python code into your app_min.py
4. **Upload HTML template** - Put hagolem_wiki.html in templates folder
5. **Reload web app** - Visit /hagolem/wiki to see it working

Detailed instructions in INSTALLATION_GUIDE.txt

## Database Structure (Simple):

### Main Table: hagolem_ideas
- Basic info: title, summary, content
- Organization: type, category, priority, status
- Tracking: views, version, dates
- Source: where it came from
- Related: tags, related ideas

### Comments Table: hagolem_comments
- What was said
- Who said it
- When
- Type (comment, suggestion, question)
- Resolved or not

### History Table: hagolem_history
- Tracks all changes
- Who changed what
- When
- Why
