## WHAT IF IT DOESN'T WORK?

### Error: "Not Found" or "404"
**Problem:** Route not added correctly
**Solution:** 
- Check app_min.py has the @app.route("/hagolem/wiki") route
- Make sure you reloaded the web app
- Check for typos in the route

### Error: "TemplateNotFound: hagolem_wiki.html"
**Problem:** Template file not uploaded
**Solution:**
- Make sure `hagolem_wiki.html` is in your `templates` folder
- Check the filename matches exactly (case-sensitive)

### Link doesn't appear on page
**Problem:** HTML not added to template
**Solution:**
- Check you edited the right template file
- Make sure you saved the file
- Reload web app
